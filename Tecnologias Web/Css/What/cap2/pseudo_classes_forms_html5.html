Pseudoclasses em formulários HTML5

Registo

Os campos de preenchimento obrigatório possuem como cor de fundo o amarelo.

Os campos onde se encontra o cursor, cujo valor introduzido seja inválido, possuem uma outline vermelha.

Os campos obrigatórios onde se encontra o cursor, cujo valor introduzido seja inválido, possuem um fundo vermelho.

-

No campo Idade se o valor introduzido for inferior a 18 e superior a 99 o fundo aparece vermelho.
